# **App Name**: Professor Portfolio

## Core Features:

- Home Page: Displays a hero section, highlights key achievements, and featured research areas.
- About Page: Provides a comprehensive academic biography, career timeline, and personal academic philosophy.
- Education Page: Presents a timeline-style listing of degrees with university names, specializations, and academic achievements.
- Research & Projects Page: Showcases research areas with visually distinct cards and project cards displaying titles, roles, and funding information.
- Publications Page: Interactive publication listing with category tabs, expandable cards, search/filter functionality, and publication statistics.
- Teaching Page: Displays courses taught in organized lists, teaching philosophy, and curriculum development contributions.
- Contact Page: Displays the university address, email, phone, academic profile links and map

## Style Guidelines:

- Primary color: Deep Navy Blue (#1a365d) for a professional and trustworthy feel.
- Background: Clean white (#FFFFFF) with subtle light gray (#F0F0F0) sections for visual separation.
- Accent: Light Blue (#ADD8E6) as a complementary accent color to the primary navy blue.
- Body font: 'Inter', a sans-serif font for a modern and readable feel.
- Headline font: 'Literata', a serif font to complement the body text with an elegant feel.
- Subtle fade-in animations on scroll and smooth page transitions for a refined user experience.
- Consistent layout across all pages, with a fixed header and a clean, structured design for easy navigation.
